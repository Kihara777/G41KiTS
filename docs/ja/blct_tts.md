# blct_tts
[中文](../zh/blct_tts.md) | [English](../en/blct_tts.md) | 日本語

blivechat カスタムテンプレート — TTS 音声読み上げ + デフォルト風ビジュアル表示

## 基本情報

| 項目 | 値 |
|------|-----|
| 種類 | blc_template |
| 依存 | blc |
| compose | none |
| バージョン | 1.2.0 |
| 作者 | 狐莉家的小爪 |

## 機能

- **ビジュアル表示** — YouTube 風チャット表示：アバター、ユーザー名、バッジ、ギフトアイコン、スパチャ強調
- **音声読み上げ** — Web Speech API TTS、外部依存なし
- **言語自動判定** — 日本語（仮名→ja-JP）を自動検出、中国語エンジンにフォールバック
- **スマートキュー** — コメント/SC は順次再生、ギフト/メンバーシップは独立キューで割り込み可能
- **中断リプレイ** — 割り込まれたコメントを優先キュー消化後に自動再読上げ
- **右クリック設定** — ページ内でレート/ピッチ/音量/フィルターをリアルタイム調整、自動保存

## インストール

```bash
./g41.sh kits add blct_tts
```

`blc_template/` ディレクトリが `.lc/custom_public/templates/blct_tts/` に展開されます。blc コンテナを再起動するとテンプレートが表示されます。

## 使い方

blivechat 設定で「TTS音声播報模板」を選択し、OBS にブラウザソースを追加。ページ上で右クリック → TTS パラメータ調整。
